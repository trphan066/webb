Danh sách các thẻ HTML

Thẻ Ý nghĩa
HTML BasicBasic cung cấp các khái niệm cơ bản về HTML, cách tạo một trang HTML đơn giản.
HTML elements giải thích các phân tử HTML và cách chúng vận hành đuọc cấu trúc trong trang web.
HTML Attibutes Giới thiệu thuôc tính của thẻ HTML giúp điểu chỉnh hiển thị và chức năng của chúng.
HTML headings dung để tạo tiêu đề với các thẻ.
HTML paragraph dung để tạo đoạn văn với thẻ.
HTML Styles áp dụng CSS trực tiếp trong HTML để tạo kiểu cho các phần tử.
HTML Formatting Cung cấp các thẻ định dạng văn bản.
HTML Quotations Dùng để hiển thị trích dẫn
HTML Comments Sử dụng thẻ để ghi chú trong mã HTML.
HTML Colors Hướng dẫn cách áp dụng màu sắc bằng mã màu HEX, RGB, hoặc tên màu.
HTML Links Dùng để tạo liên kết với thẻ
HTML Images Chèn hình ảnh vào trang web bằng thẻ
HTML Tables Tạo bảng bằng thẻ và các thẻ liên quan
HTML Lists Tạo danh sách có thứ tự và không thứ tự , cùng các mục danh sách
HTML Block & Inline Giới thiệu sự khác biệt giữa phần tử khối (block) và phần tử nội dòng (inline).
